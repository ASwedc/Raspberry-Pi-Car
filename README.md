# Raspberry-Pi-Car
需要安裝Flask才能使用

Flask資料庫 

http://flask.pocoo.org/docs/0.12/

安裝指令
```
   $ sudo pip install Flask
```
===================
程式的執行方式打開command視窗
先將目錄移動到程式下載的位置

之後輸入指令
```
   $ sudo python app.py
```
